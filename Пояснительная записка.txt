Пояснительная записка к проекту "WildRoutes"
Платформа для поиска и бронирования авторских путешествий

1. Описание проекта
WildRoutes — это современная веб-платформа, предназначенная для объединения
организаторов уникальных авторских туров (гидов) и путешественников, ищущих нестандартные
маршруты. Проект решает проблему поиска проверенных частных гидов и упрощает процесс
бронирования активного отдыха.

2. Основной функционал
Витрина туров: Каталог с системой фильтрации по категориям (Горы, Лес, Экстрим,
Релакс).
Личный кабинет: Разграничение ролей для туристов (просмотр заявок) и гидов
(управление своими турами).
Система бронирования: Возможность подачи заявки на участие в туре с отслеживанием
статусов (Ожидание, Подтверждено, Отменено).
Интеграция ИИ: Чат-бот на базе модели nvidia: nemotron для помощи в составлении
маршрутов.
Интерактивные карты: Использование Яндекс.Карт для отображения точек сбора и
маршрутов.

3. Технологический стек
Для реализации проекта были выбраны следующие технологии:
Backend: Python 3.x, Flask (основной фреймворк).
База данных: SQLAlchemy 2.0 (ORM) для управления реляционными данными.
Frontend: HTML5, CSS3 (дизайн), JS для анимаций и переходов.
Безопасность: Flask-Login для авторизации и Flask-WTF для защиты форм.
Внешние API: OpenRouter (OpenAI API) и Яндекс.Карты.

4. Инструкция по запуску проекта:
    1. Загрузить проект с GitHub
        Клонировать репозиторий https://github.com/PaschencoDima/WildRoutes-by-CCC
        git clone https://github.com/PaschencoDima/WildRoutes-by-CCC
    2. Установка зависимостей
        Установите все необходимые библиотеки, включая Flask, SQLAlchemy и Flask-Login:
        pip install -r requirements.txt
    3. Запуск
        Запустить файл main.py
        python main.py


5. Уникальные особенности (Коллаборация)
Одной из ключевых фишек проекта является интеграция с партнерской кофейней. Кофейня
выступает как точка сбора участников перед походом.
Бонус от спонсора: Показав приложение в одной из кофеен вы получите бесплатный напиток на ваш выбор(но стоит проверять наличие напитка)

6. Дизайн и интерфейс
Интерфейс выполнен в премиальной темной теме с использованием природных оттенков:
песочный и акцентный кофейный (янтарь/охра).
Это придает приложению современный и глубокий вид.

7. Заключение
Проект WildRoutes представляет собой законченное MVP-решение (Minimum Viable Product),
готовое к масштабированию. Сочетание функционала социальной платформы, сервиса
бронирования и системы лояльности через оффлайн-партнеров делает его конкурентоспособным
на рынке авторского туризма.
